Core Commands

Everything you need to build features

/piv-speckit:plan-feature Plan a feature with auto-context loading (creates spec, plan, tasks, quickstart artifacts)
/piv-speckit:execute Implement the plan following strict TDD cycle
/piv-speckit:validate Run comprehensive validation (compilation, tests, coverage, quality)
/piv-speckit:prime Manual context refresh (optional ‚Äî auto-prime happens on plan-feature)
/piv-speckit:code-review Technical code review for quality and bugs
/piv-speckit:learn Extract learnings from code reviews to improve future implementations
